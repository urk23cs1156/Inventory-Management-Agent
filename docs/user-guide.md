# User Guide

## Getting Started

1. Launch the application using Flask
2. Open the application in a web browser
3. Navigate to the Inventory Dashboard

## Adding Inventory Items

1. Click on the "Add Item" button
2. Enter the item name
3. Enter the current quantity
4. Enter the minimum threshold
5. Submit the form

## Viewing Inventory

- All items are displayed on the dashboard
- Quantity and threshold values are shown clearly
- Stock status is visually indicated

## Viewing Restock Recommendations

1. Click on "Restock Recommendations"
2. The system displays items below threshold
3. Recommended restock quantities are shown
4. Recommendations are generated by the intelligent agent
